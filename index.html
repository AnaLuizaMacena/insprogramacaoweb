Sou estudante de Licenciatura em Computação no Instituto Federal do Piauí - Campus Zona Sul, e este projeto faz parte das atividades propostas na disciplina "Introdução à Programação Web", ministrada pelo Professor Msc. Jefferson de Sousa Silva. O principal objetivo deste trabalho é:

  • Estruturar documentos web utilizando HTML;
  • Aplicar as melhores práticas e padrões da web recomendados pelo W3C;
  • Utilizar CSS para estilizar e formatar a apresentação dos documentos;
  • Implementar funcionalidades interativas com JavaScript;
  • Desenvolver sites responsivos que se adaptem a diferentes dispositivos.
